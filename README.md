## 🏗️ How I Built It
 
The project was split across **3 days (~4 hours total)**:
 
**Day 1 — Layout**
> Focused entirely on structure. Built the calculator UI using React components and vanilla CSS — buttons, display screen, and overall layout — with no functionality attached yet.
 
**Day 2 & 3 — Functionality**
> Wired up all the logic: handling number input, chaining operations, computing results, and clearing state. This is where `useState`, props, and handler functions came together.
 
The deliberate split between *layout first, logic second* kept things manageable and helped me understand how React re-renders work in practice.
 
---
 
## 🎓 What I Learned
 
Building this project from scratch — **without any AI assistance** — was the real challenge, and the real reward.
 
- 🔁 **`useState`** — understood how state drives UI updates in real-time as numbers and operators change
- 📦 **Props** — learned how to pass data and functions between parent and child components cleanly
- ⚙️ **Functions in React** — got comfortable writing event handlers and connecting them to JSX elements
- 🧠 **Problem-solving** — figuring out operator chaining and display logic independently built genuine confidence
